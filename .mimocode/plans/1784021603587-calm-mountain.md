# План: Адаптация каталога Товары для Производств и Компаний

## Цель
Компании и производства смогут создавать карточки товаров с ценами и описанием. Клиенты смогут выбирать товары по цене, видеть от какой компании, и добавлять в избранное.

## Что нужно сделать

### 1. Добавить избранное для товаров

**FavoriteButton** (`src/components/FavoriteButton.tsx`):
- Добавить тип `'product'` в интерфейс

**Favorites API** (`src/app/api/favorites/route.ts`):
- Добавить загрузку товаров из `prisma.product` по `productIds`

**Favorites страница** (`src/app/dashboard/favorites/page.tsx`):
- Добавить отображение избранных товаров

### 2. Добавить manufacturerId в Product

**Schema** (`prisma/schema.prisma`):
```prisma
model Product {

  manufacturer   Manufacturer?  @relation(fields: [manufacturerId], references: [id])

}
```

## Верификация

1. `npx prisma db push` — миграция
2. `npm run build` — без ошибок
3. Ручное тестирование:
   - Компания создаёт товар → появляется в каталоге
   - Клиент видит логотип компании в карточке
   - Клиент добавляет товар в избранное → видит в разделе "Избранное"
   - Фильтрация по цене, категории, производителю работает
