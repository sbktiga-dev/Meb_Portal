# План улучшения чата

## Текущее состояние

**Модель Message** (`prisma/schema.prisma:415`):
- `content: String` — только текст
- Нет полей для файлов/медиа

**UI чата** (`src/app/dashboard/messages/[id]/page.tsx`):
- Простой текстовый input
- Нет загрузки файлов
- Нет эмодзи
- Нет предпросмотра медиа

**Upload API** (`src/app/api/upload/route.ts`):
- Уже поддерживает: изображения (JPG, PNG, GIF, WebP), видео (MP4, WebM), документы
- Загружает в Yandex Object Storage

---

## Что нужно сделать

### 1. Миграция базы данных — добавить поле attachments в Message

**Файл:** `prisma/schema.prisma`

```prisma
model Message {
  id            String         @id @default(cuid())
  content       String
  attachments   String         @default("[]")  // JSON массив URL файлов
  createdAt     DateTime       @default(now())
  author        User           @relation(...)
  authorId      String
  conversation  Conversation   @relation(...)
  conversationId String
}
```

После: `npx prisma db push`

### 2. API — поддержка attachments в отправке сообщений

**Файл:** `src/app/api/conversations/[id]/messages/route.ts`

Изменения в POST:
- Принимать `attachments` (массив URL) из body
- Сохранять в Message.attachments
- Валидация: макс. 5 файлов, макс. 10 МБ каждый

### 3. Компонент EmojiPicker

**Новый файл:** `src/components/EmojiPicker.tsx`

- Сетка популярных эмодзи (как в мессенджерах)
- Категории: Смайлики, Жесты, Животные, Еда, Объекты
- Поиск по эмодзи
- Закрытие по клику снаружи

### 4. Обновление UI чата

**Файл:** `src/app/dashboard/messages/[id]/page.tsx`

Изменения:
- **Кнопка прикрепления файла** (слева от input)
  - Иконка скрепки → выбор файла → загрузка через /api/upload → добавление URL в attachments
  - Поддержка drag-and-drop
- **Кнопка эмодзи** (справа от input)
  - Открывает EmojiPicker → вставляет эмодзи в текст
- **Отображение медиа в сообщениях**
  - Изображения → кликабельные превью (открываются в лайтбоксе)
  - Видео → встроенный плеер
  - Документы → ссылки на скачивание
- **Индикатор загрузки** файла

### 5. Компонент MessageAttachments

**Новый файл:** `src/components/MessageAttachments.tsx`

- Рендерит вложения в сообщении
- Изображения: `<Image>` с превью, клик → лайтбокс
- Видео: `<video>` с контролами
- Документы: ссылка с иконкой типа файла

### 6. Отправка из избранного (опционально)

**Реализация:** Кнопка "Из избранного" открывает модалку с избранными изображениями пользователя. Выбранное изображение прикрепляется как attachment.

**Файл:** `src/components/FavoritesPicker.tsx`
- Загружает избранные изображения пользователя
- Сетка превью
- Выбор → добавление как attachment
