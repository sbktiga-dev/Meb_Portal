## Формат данных для ГОСТов
Reference.content будет содержать JSON с типом:
- `{"_type": "text", "sections": [{"title": "...", "text": "..."}]}` для длинного текста
- `{"ключ": ["значение"]}` для таблиц (существующий формат)
