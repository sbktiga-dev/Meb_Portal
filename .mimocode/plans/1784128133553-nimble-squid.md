# План: Заглушки баннеров на всех страницах

## Задача
Добавить пустые слоты-заглушки на все страницы с баннерами. Формат книжный (вертикальный).

## Страницы

### 1. Лента — `src/app/feed/page.tsx`
**5 заглушек слева + 5 справа**

- Текущая сетка: `grid-cols-[240px_1fr_240px]` → оставить, aside всегда видны
- Левая колонка: 5 заглушек (sticky)
- Правая колонка: 5 заглушек (sticky)
- Когда есть реальные баннеры — заглушки заменяются
- На `<lg` — колонки скрыты

### 2. Каталог изображений — `src/app/gallery/page.tsx`
**Добавить сайдбары + заглушку в контенте**

- Обернуть в 3-колоночную сетку (как лента)
- Левая: 2 заглушки, правая: 2 заглушки
- Контентный баннер — заглушка когда пусто
- На `<lg` — без сайдбаров

### 3. Каталог товаров — `src/app/products/page.tsx`
**3 заглушки слева + 3 справа**

- Текущая сетка: просто `grid md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4`
- Обернуть в 3-колоночную сетку с aside
- Левая: 3 заглушки (sticky)
- Правая: 3 заглушки (sticky)
- На `<lg` — без сайдбаров

### 4. Профиль — `src/app/profile/[userId]/page.tsx`
**Без изменений** — уже имеет заглушки через `ProfileSideBanner`.
