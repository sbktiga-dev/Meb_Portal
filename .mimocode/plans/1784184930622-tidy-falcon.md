# План: баннеры на странице Каталога (gallery)

## Что делаем
Привести layout баннеров на странице `gallery` к такому же виду как на `feed`:
- Баннеры с отступом 5мм от краёв страницы
- Ширина колонок баннеров: 320px
- Отступ между баннером и постами: 20мм (gap-[76px])

## Файл
`src/app/gallery/page.tsx`

## Изменения

### 1. Вынести сетку за пределы `section-container`

Сейчас (строка 244): сетка с баннерами находится ВНУТРИ `section-container py-10`, который добавляет `px-[10mm]` отступ.

Нужно:
- Закрыть `section-container` ДО сетки (после блока фильтров/поиска, строка ~235)
- Обернуть сетку в `<div className="px-[5mm]">`

### 2. Изменить grid-класс

Текущий (строка 244):
```
grid grid-cols-1 lg:grid-cols-[256px_1fr_256px] gap-6
```

Новый (как в feed):
```
grid grid-cols-1 lg:grid-cols-[320px_1fr_320px] gap-[76px]
```

### 3. Закрыть обёртку

После закрытия grid (`</div>`) добавить закрытие `px-[5mm]` обёртки.

## Верификация
1. TypeScript: `npx tsc --noEmit` — ошибок в gallery/page.tsx быть не должно
2. Визуально: баннеры-заглушки с отступом 5мм от краёв, 320px шириной, 20мм до центрального контента
3. На мобильных: баннеры скрыты (hidden lg:block)
