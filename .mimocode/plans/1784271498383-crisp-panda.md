# Акции от МебПортал

## Суть
Отдельная страница "Акции от МебПортал" с ограниченными по времени предложениями. Каждая акция имеет условия, сроки и кнопку участия.

---

## Акции

### 1. Скидка за Виджет (постоянная)
- **Условие:** Разместить виджет МебПортала на своём сайте
- **Награда:** Скидка 15% на подписку (Lite/Pro/Premium)
- **Как участвовать:** Установить виджет → загрузить скриншот + ссылку на сайт → администратор проверяет → начисляется скидка
- **Срок:** Постоянная акция

### 2. Реферальная программа (постоянная)
- **Условие:** Пригласить 3 коллег через реферальную ссылку
- **Награда:** 1 месяц PRO бесплатно
- **Как участвовать:** Поделиться ссылкой → 3 друга регистрируются → автоматически
- **Срок:** Постоянная акция

### 3. Заполни профиль (постоянная)
- **Условие:** Заполнить профиль на 100% (фото, описание, город, сайт, соцсети)
- **Награда:** Бейдж "Проверенный" + кредит 500₽ на продвижение
- **Как участвовать:** Заполнить все поля → автоматическая проверка
- **Срок:** Постоянная акция

### 4. Активный автор (постоянная)
- **Условие:** Опубликовать 10 постов за месяц
- **Награда:** 1 бесплатный баннер
- **Как участвовать:** Писать посты → автоматический подсчёт
- **Срок:** Постоянная акция

### 5. Отзывы в награду (постоянная)
- **Условие:** Оставить 5 одобренных отзывов
- **Награда:** 1 месяц Premium бесплатно
- **Как участвовать:** Писать отзывы → после одобрения 5 штук → автоматически
- **Срок:** Постоянная акция

### 6. Первый пост (для новичков)
- **Условие:** Опубликовать первый пост на платформе
- **Награда:** Бесплатное продвижение первого поста (закрепление в ленте на 7 дней)
- **Как участвовать:** Опубликовать пост → автоматически
- **Срок:** Для новых пользователей (первые 30 дней)

### 7. Сезонные акции (по 3 месяца)
- **Лето с МебПорталом** (июнь-август): Скидка 20% на Premium
- **Осень для бизнеса** (сентябрь-ноябрь): 2 баннера по цене 1
- **Новогодний подарок** (декабрь-февраль): 1 месяц Premium при оплате года
- **Весеннее обновление** (март-май): Бесплатный виджет для всех

## Реализация

### Новые файлы

**`src/app/promos/page.tsx`** — страница акций
- Сетка карточек акций (2-3 колонки)
- Каждая карточка: иконка, название, описание, условия, сроки, статус (активна/завершена), кнопка "Участвовать"
- Фильтр: все / активные / завершённые

**`src/app/api/promos/route.ts`** — GET: список всех акций + статус участия текущего пользователя

**`src/app/api/promos/join/route.ts`** — POST: участие в акции (для акций с ручной проверкой — виджет)

**`src/app/admin/promos/page.tsx`** — страница проверки заявок на акции (только для админа)
- Список заявок на "Скидку за виджет"
- Просмотр скриншотов и ссылок
- Одобрение/отклонение

### Изменения в БД

**`prisma/schema.prisma`** — новая модель:

```prisma
model PromoParticipation {
  id          String   @id @default(cuid())
  promoKey    String   // ключ акции: widget_discount, referral, profile_100, active_author, reviews_reward, first_post
  status      String   @default("pending") // pending | approved | rejected | completed
  proofUrl    String?  // ссылка на сайт (для виджета)
  proofImages String   @default("[]") // JSON массив URL скриншотов
  note        String?  // комментарий пользователя
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  user        User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  userId      String

  @@unique([userId, promoKey])
  @@index([promoKey, status])
}
```

### Изменения в навигации

**`src/components/Sidebar.tsx`** — добавить пункт "Акции от МебПортал" (с иконкой подарка)

**`src/components/Header.tsx`** — добавить ссылку "Акции" в навигацию (или в выпадающее меню "Участники")

### Данные акций

Хранить в `src/lib/promos.ts` как массив объектов:

```typescript
interface Promo {
  key: string;
  title: string;
  description: string;
  icon: string;
  reward: string;
  conditions: string[];
  type: 'auto' | 'manual'; // auto = проверяется автоматически, manual = админ проверяет
  startDate?: string;
  endDate?: string;
  permanent: boolean;
}
```

## Проверка

1. `/promos` — страница открывается, все акции отображаются
2. Нажать "Участвовать" на виджете → загрузка скриншотов
3. Админ видит заявку → проверяет → одобряет
4. Автоматические акции (профиль, посты) → проверяются на сервере
5. Сезонные акции — отображаются в правильные месяцы
