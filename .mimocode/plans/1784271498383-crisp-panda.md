# Баннер Панорама + Мини Банер + Ротация

## Суть

**Панорама** — один длинный баннер, состоящий из 5 картинок в ряд, разделённых тонкими полосками. Пользователь загружает 5 фото — они отображаются как единая панорамная полоса.

**Мини Банер** — маленький компактный баннер с одной картинкой.

**Ротация:**
- Пока баннеров <= слотов — баннеры заполняют слоты поочерёдно
- Когда баннеров > слотов — ротация каждые 10 секунд с плавной fade-анимацией

---

## 1. Миграция БД

**`prisma/schema.prisma`** — модель Banner:

```prisma
bannerType     String   @default("standard") // standard | panorama | mini
images         String   @default("[]")       // JSON массив URL для панорамы (5 фото)
```

## 2. Компоненты

**`src/components/BannerPanorama.tsx`** — панорама
- Пропсы: `images: string[]` (5 URL), `title: string`, `linkUrl: string`
- 5 картинок в flex-ряд, `gap-1` (4px серые полоски между ними)
- Каждая картинка: `flex-1 aspect-[4/3] object-cover rounded-sm`
- Hover: `scale-105` на каждой с transition
- Бейдж "Реклама" в левом верхнем углу
- Обёрнут в `<Link>`

**`src/components/BannerMini.tsx`** — мини баннер
- Пропсы: `imageUrl: string`, `title: string`, `linkUrl: string`
- Высота: `min-h-[160px]` (вместо 337 у стандартного)
- Бейдж "Реклама" компактный
- Обёрнут в `<Link>`

**`src/components/BannerRotator.tsx`** — ротатор с fade-анимацией
- Пропсы: `banners: any[]`, `type: 'mini' | 'panorama'`, `slots: number`, `interval?: number` (10000ms)
- Логика:
  - Если banners.length <= slots → показать все сразу (сетка)
  - Если banners.length > slots → показывать текущую "страницу" из `slots` баннеров, переключать каждые 10 секунд
- Анимация: `opacity` transition 300ms при смене
- Стейт: `currentIndex` (начало текущей страницы)
- useEffect с setInterval для таймера

## 3. Форма создания

**`src/app/dashboard/promotion/page.tsx`**:

Добавить выбор типа баннера (3 кнопки):
- Стандартный (по умолчанию) — 1 картинка
- Панорама — 5 картинок в ряд
- Мини баннер — 1 картинка, маленький

Для **панорамы**: 5 слотов загрузки (горизонтальная сетка), валидация что все 5 заполнены.

---

## 4. API

**`src/app/api/promotion/banners/route.ts`**:
- POST: принять `bannerType` + `images[]` (5 URL для panorama)
- Валидация: panorama → images.length === 5

---

## 5. Отображение

**`src/app/gallery/page.tsx`** и **`src/app/feed/page.tsx`**:

Заменить статичные баннеры на ротаторы:
- Слева/справа: `<BannerRotator type="mini" banners={miniBanners} slots={1} />`
- Сверху: `<BannerRotator type="panorama" banners={panoramaBanners} slots={1} />`
