## 1. Приветственный модал (WelcomeModal)

### Новый файл: `src/components/WelcomeModal.tsx`

**Логика:**
- На mount: fetch `/api/auth/me` → определяем роль
- Проверяем `localStorage.getItem('welcome_seen_{role}')`
- Если нет — показываем модал
- Закрытие: `localStorage.setItem('welcome_seen_{role}', '1')`

**Содержимое по ролям:**

| Роль | Заголовок | Описание |
|---|---|---|
| ADMIN | Добро пожаловать, Администратор! | Управляйте пользователями, контентом и настройками через панель «Админ». |
| USER | Добро пожаловать! | Создавайте публикации, добавляйте товары и делитесь работами с коллегами. |
| COMPANY | Добро пожаловать! | Профиль компании готов. Заполните информацию и найдите поставщиков. |
| SUPPLIER | Добро пожаловать! | Профиль поставщика создан. Добавляйте прайс-листы и находите клиентов. |
| MANUFACTURER | Добро пожаловать! | Представьте производство, разместите каталог и наладите контакты. |
| CLIENT | Добро пожаловать! | Просматривайте каталог, скачивайте документы и находите поставщиков. |

**Дизайн:**
- Оверлей: `fixed inset-0 bg-black/50 z-[200] backdrop-blur-sm`
- Карточка: `bg-white dark:bg-gray-800 rounded-2xl p-8 max-w-md shadow-2xl`
- Градиентный круг с ролью, заголовок, текст, кнопка «Понятно», крестик

**Интеграция:** добавить `<WelcomeModal />` в `HomeContent.tsx`

## 3. Growth-механики

### 3.1 SEO-оптимизация публичных профилей

**Изменить: `src/app/profile/[userId]/page.tsx`**
- Добавить динамический `Metadata` (title, description, OpenGraph)
- Пример: `<title>{user.name} — МебПортал</title>`, description на основе профиля
- Добавить JSON-LD structured data для поисковиков
- canonical URL

**Изменить: `src/app/suppliers/[id]/page.tsx`, `src/app/companies/[id]/page.tsx`**
- Аналогичная SEO-оптимизация для страниц поставщиков и компаний

### 3.2 Кнопки шаринга (ShareButtons)

**Новый файл: `src/components/ShareButtons.tsx`**

Компонент с кнопками:
- VK (через `https://vk.com/share.php?url=...`)
- Telegram (через `https://t.me/share/url?url=...`)
- WhatsApp (через `https://wa.me/?text=...`)
- Копирование ссылки (clipboard API)

Props: `url: string`, `title: string`, `compact?: boolean`

**Интеграция в:**
- Публикации ленты (`/feed/[id]`)
- Товары (`/products/[id]`)
- Профили пользователей
- Портфолио

### 3.3 Реферальная система

**Изменить: `prisma/schema.prisma`** — добавить модель:
```prisma
model Referral {
  id          String   @id @default(cuid())
  referrerId  String
  referredId  String   @unique
  createdAt   DateTime @default(now())
  referrer    User     @relation(fields: [referrerId], references: [id])
  referred    User     @relation(fields: [referredId], references: [id])
}
```

**Новый API: `src/app/api/referral/route.ts`**
- `GET /api/referral` — получить уникальную реферальную ссылку текущего пользователя
- `POST /api/referral` — применить реферальный код при регистрации

**Изменить: `src/app/register/page.tsx`**
- Поле "Реферальный код" (опциональное)
- При регистрации с кодом → создаём запись Referral

**Новый файл: `src/components/ReferralBanner.tsx`**
- Баннер в личном кабинете: "Пригласите коллегу — получите месяц PRO"
- Копирование реферальной ссылки
- Счётчик приглашённых

**Интеграция:** добавить `<ReferralBanner />` в `src/app/dashboard/page.tsx`

### 3.4 Виджет "Мы на МебПортале"
