# ПЛАН РАЗРАБОТКИ «МЕБЕЛЬНЫЙ ПОРТАЛ»

## Концепция

Платформа для мебельной отрасли: бесплатная библиотека изображений, полезные документы, каталог поставщиков и карточки компаний. Привлечение специалистов через контент → монетизация через поставщиков и рекламу.

---

## ТИПЫ АККАУНТОВ

| Тип | Возможности | Монетизация |
|-----|------------|-------------|
| **ИП / Компания** | Карточка компании, реклама в каталоге, приоритет в поиске | Платная подписка / размещение |
| **Поставщик** | Каталог товаров (фурнитура, ЛДСП, техника), прайс-листы | Платное размещение товара |
| **Производство** | Портфолио работ, технические возможности, география | Платное продвижение |
| **Специалист** | Менеджер, дизайнер, технолог, установщик — личный профиль с рейтингом | Бесплатно (привлечение аудитории) |

---

## РАЗДЕЛ «ПОЛЕЗНЫЕ МАТЕРИАЛЫ»

### Документы
- Шаблоны договоров (подряд, поставка, гарантия)
- Акты выполненных работ
- Счета, спецификации, коммерческие предложения
- Технические задания на замер

### Техническая информация
- Стандарты размеров (посудомоечная машина, мойка, варочная панель, духовка)
- Таблицы кромкования и сверловки
- Нормы расхода материалов
- Паспорта на фурнитуру (Blum, Hettich, Grass)
- Чертежи и схемы установки

### Статьи и гайды
- Как составить смету на кухню
- Расчёт столешницы: формулы
- Выбор фасадов: сравнение материалов
- Маркетинг для мебельщиков

## ДОРОЖНАЯ КАРТА

### Фаза 1: MVP (месяцы 1–2)
- [ ] Сайт с регистрацией (по ИНН для верификации)
- [ ] Фотобиблиотека (200–300 изображений)
- [ ] Галерея с фильтрами (стиль, тип, размер)
- [ ] Скачивание в высоком разрешении
- [ ] Раздел документов (3–5 шаблонов)
- [ ] Технические справочники
- [ ] Личный кабинет с историей загрузок
- [ ] Роль «Специалист» (менеджер, дизайнер, технолог, установщик)
- **Цель**: 50–100 пользователей

### Фаза 2: Рост (месяцы 3–4)
- [ ] Карточки компаний (ИП)
- [ ] Раздел поставщиков (10–20 компаний)
- [ ] Раздел «Производства»
- [ ] Рейтинг специалистов
- [ ] SEO-оптимизация
- [ ] Telegram-канал
- **Цель**: 200–300 пользователей

### Фаза 3: Монетизация (месяцы 5–6)
- [ ] Платное размещение поставщиков
- [ ] Рекламные баннеры для ИП
- [ ] Реферальная программа
- [ ] Email-рассылки
- **Цель**: 500–700 пользователей

### Фаза 4: Масштабирование (месяцы 7–12)
- [ ] Интеграция с CRM (МебельСофт, KD Студия)
- [ ] API для мебельных систем
- [ ] Мобильное приложение
- [ ] Премиум-подписка
- [ ] Гео-расширение (Беларусь, Казахстан, Узбекистан)
- **Цель**: 1000+ пользователей

## СТРАТЕГИЯ ПРИВЛЕЧЕНИЯ

### Органический рост
- **SEO**: Ключевые запросы:
  - «каталог кухонь для клиентов»
  - «готовые фото мебели для сайта»
  - «изображения корпусной мебели бесплатно»
  - «шаблон договора на мебель»
- **Контент**: 2 статьи в неделю (гайды, обзоры стилей)
- **Форумы**: Ответы на вопросах на mebelmasteronline.ru и др.

### Telegram
- Ежедневные примеры из библиотеки
- Партнёрства с чатами мебельщиков
- Обзоры и рекламные посты

### Реферальная программа
- «Пригласи 3 коллег — получи доступ к премиум-коллекции»

### Вебинары
- Бесплатные вебинары «Как составить каталог из наших фото»

### Привлечение поставщиков
- **Бесплатный пилот**: Первые 3 месяца бесплатно для топ-20 поставщиков
- **Кейсы**: «Как поставщик получил 50 заявок за месяц»
- **Партнёрство с выставками**: Мебельные выставки и форумы
- **Прямые продажи**: Холодные звонки по базе поставщиков

## ТЕХНИЧЕСКИЙ СТЕК (рекомендация)

- **Frontend**: React / Next.js (SSR для SEO)
- **Backend**: Node.js / FastAPI
- **БД**: PostgreSQL + Redis (кэширование)
- **Хранилище файлов**: MinIO / S3 (изображения, документы)
- **Поиск**: Elasticsearch / Meilisearch
- **Авторизация**: JWT + OAuth2 (Email, ИНН верификация)
- **Хостинг**: Yandex Cloud / Timeweb / Selectel

---

## КОНКУРЕНТНЫЕ ПРЕИМУЩЕСТВА

1. **Узкая ниша** — только мебельная отрасль (не generic фотосток)
2. **Постоянное пополнение** — ваши проекты как источник контента
3. **Понимание потребностей** — реальные задачи мебельщиков
4. **Экосистема** — поставщики + производители + специалисты в одном месте
5. **Бесплатный доступ** — быстрый набор аудитории

## КРИТИЧЕСКИЕ ФАКТОРЫ УСПЕХА

- Качество контента (только профессиональные фото)
- Скорость загрузки сайта
- Оперативное пополнение базы (20–30 новых изображений в неделю)
- Отклик на обратную связь пользователей
- Регулярное добавление документов и справочников
- Привлечение и удержание поставщиков
