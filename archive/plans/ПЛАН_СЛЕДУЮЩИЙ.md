# ПЛАН: Лента новостей + Портфолио для участников

## Концепция

Мебельный портал превращается в **сообщество** — у каждого участника (мебельщик, поставщик, специалист, производитель) есть профиль с портфолио, а на главной работает **лента новостей** (как соцсеть), где участники публикуют посты: проекты, фото работ, статьи, товары.

---

## ЭТАП 1: Модель данных (Prisma)

**Файл: `prisma/schema.prisma`**

```
Post {
  id            String   @id @default(cuid())
  authorType    String   // "user" | "supplier" | "specialist" | "company" | "manufacturer"
  authorId      String
  title         String
  content       String   // markdown/text
  images        String   @default("[]")  // JSON array of image URLs
  category      String?  // "project" | "article" | "product" | "news"
  tags          String   @default("[]")  // JSON array
  likes         Int      @default(0)
  comments      Comment[]
  views         Int      @default(0)
  isPublished   Boolean  @default(true)
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
}

Comment {
  id        String   @id @default(cuid())
  postId    String
  post      Post     @relation(fields: [postId], references: [id], onDelete: Cascade)
  authorType String
  authorId  String
  authorName String
  content   String
  createdAt DateTime @default(now())
}

PortfolioItem {
  id          String   @id @default(cuid())
  authorType  String   // "user" | "supplier" | "specialist" | "company" | "manufacturer"
  authorId    String
  title       String
  description String?
  images      String   @default("[]")  // JSON array
  category    String?  // "kitchen" | "bedroom" | "office" | etc.
  tags        String   @default("[]")
  createdAt   DateTime @default(now())
}
```

**API:**
- `POST/GET /api/posts` — CRUD постов
- `GET /api/feed` — лента для главной (с пагинацией, фильтрацией)
- `POST /api/posts/[id]/like` — лайк
- `POST /api/posts/[id]/comments` — комментарий
- `GET/POST/DELETE /api/portfolio` — CRUD портфолио участника
- `GET /api/portfolio/[authorType]/[authorId]` — портфолио конкретного участника

## Ожидаемый результат

1. Главная страница с **лентой новостей** (как лента в соцсети)

4. Портфолио **публикуется** и доступно через профиль
5. Взаимодействие: **лайки, комментарии, обсуждения**
