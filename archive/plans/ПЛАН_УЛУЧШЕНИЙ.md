# План улучшений — Фаза 3

## Приоритет 1: Лента на главном экране

**Текущее состояние**: ✅ Реализовано
- Показывает только заглушки (нет реальных фото в постах)
- Нет превью изображений постов
- Нет возможности лайкать прямо с главной

### Шаг 1.1: Улучшить карточки постов на главной
- Добавить реальные изображения из `images` JSON-поля поста
- Показывать фото-галерею (2-3 миниатюры) вместо плейсхолдера
- Добавить кнопку лайка прямо на карточке (inline, без перехода)
- Показать avatar пользователя вместо инициала

### Шаг 1.2: Добавить фильтрацию ленты
- Табы: Все / Новости / Проекты / Статьи / Товары
- Сортировка: Новые / Популярные / Обсуждаемые
- Превью-аватар автора + название компании (если есть)

### Шаг 1.3: Лента в sidebar (опционально)
- Мини-лента в правом sidebar на десктопе
- 3-5 последних постов с аватарами и лайками
