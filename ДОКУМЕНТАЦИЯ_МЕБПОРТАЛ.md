# МебПортал — Библиотека мебельщика

## Описание проекта

**МебПортал** — современная онлайн-платформа для профессионалов мебельной индустрии. Объединяет поставщиков, производителей, дизайнеров, технологов и клиентов в едином информационном пространстве.

**Сайт:** [www.mebportal.online](https://www.mebportal.online)

**Стек технологий:**
- Next.js 14 (App Router) + TypeScript
- PostgreSQL (Neon) + Prisma ORM
- Tailwind CSS
- Vercel (хостинг) + Vercel Blob Storage (файлы)

---

## Целевая аудитория

- **Дизайнеры** — создание и демонстрация проектов
- **Технологи** — обмен опытом и документами
- **Производства** — поиск поставщиков и партнёров
- **Поставщики** — продвижение товаров и услуг
- **Клиенты** — поиск специалистов и заказ мебели

- Публикации с фото и видео

- Фильтрация по категориям (Новости, Проекты, Статьи, Товары)

- Сортировка: новые, популярные, обсуждаемые

## Монетизация — Система подписок

### Тариф Lite (1 500 ₽/мес | 15 000 ₽/год)
- Продвижение постов в ленте
- 1 баннер в ленте или каталоге
- Метка «Рекомендовано» на постах

### Тариф Pro (4 000 ₽/мес | 40 000 ₽/год)
- Всё из Lite
- До 2 баннеров в неделю
- Приоритет в каталогах (показывается первым)
- Значок PRO на профиле

### Тариф Premium (8 000 ₽/мес | 80 000 ₽/год)
- Всё из Pro
- До 4 баннеров в неделю
- Максимальный приоритет в каталогах
- Значок PREMIUM на профиле
- Рекламные посты (акции) на профиле
- Аналитика профиля (просмотры, лайки, посты)

## Технические особенности

### Безопасность
- JWT-аутентификация через HTTP-only cookie
- Rate limiting для API ( dual: IP + email)
- Валидация email (только российские домены)
- Обязательная верификация email при регистрации
- Защита от XSS и SQL-инъекций

### Производительность
- Server-Side Rendering (SSR) для SEO
- Cache-Control заголовки на API
- Lazy loading изображений
- Infinite Scroll для длинных списков
-keleton-загрузка для UX

### Мобильная адаптация
- Адаптивный дизайн для всех экранов
- Нижняя панель навигации на мобильных
- Touch-оптимизация интерфейса
- Поддержка pinch-to-zoom для изображений

### Контент
- Загрузка фото (JPG, PNG, GIF, WebP)
- Загрузка видео (MP4, WebM, MOV)
- Lightbox для просмотра изображений
- Максимум 10 файлов на пост
