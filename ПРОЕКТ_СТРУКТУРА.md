# Структура проекта "Мебельный портал"

## Обзор
Next.js 14 веб-приложение — библиотека изображений мебели, документы, справочники, каталог поставщиков и производств.

## Технологии
- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **БД**: PostgreSQL (Neon) + Prisma ORM
- **Auth**: JWT (jsonwebtoken) + bcryptjs
- **Валидация**: Zod

## API Endpoints (24 группы)

### Auth
- `POST /api/auth/register` — Регистрация
- `POST /api/auth/login` — Вход
- `GET /api/auth/me` — Текущий пользователь
- `PUT /api/auth/update` — Обновление профиля

### Content
- `GET/POST /api/posts` — Посты
- `GET/PUT/DELETE /api/posts/[id]` — Пост по ID
- `POST /api/posts/[id]/like` — Лайк
- `POST /api/posts/[id]/repost` — Репост
- `GET/POST /api/posts/[id]/comments` — Комментарии
- `GET/POST /api/images` — Изображения
- `GET/POST /api/documents` — Документы
- `GET /api/refs` — Справочники

### Business
- `GET/POST /api/companies` — Компании
- `GET/POST /api/suppliers` — Поставщики
- `GET/POST /api/manufacturers` — Производители
- `GET/POST /api/specialists` — Специалисты
- `GET/POST /api/products` — Товары
- `POST /api/products/[id]/reviews` — Отзывы

### Social
- `GET/POST /api/conversations` — Диалоги
- `GET/POST /api/conversations/[id]/messages` — Сообщения
- `GET/POST /api/groups` — Группы
- `POST /api/groups/[id]/join` — Вступить в группу
- `GET/POST /api/events` — Мероприятия
- `POST /api/events/[id]/join` — Участие
- `GET/POST /api/favorites` — Избранное
- `GET/POST /api/bookmarks` — Закладки
- `GET/POST /api/notifications` — Уведомления
- `POST /api/users/[id]/follow` — Подписка
- `GET /api/users/[id]/followers` — Подписчики
- `GET /api/users/[id]/following` — Подписки

### System
- `GET /api/search` — Поиск
- `GET /api/stats` — Статистика
- `GET /api/feed` — Лента
- `GET /api/downloads` — Загрузки
- `GET /api/portfolio` — Портфолио
- `POST /api/upload` — Загрузка файлов
- `GET /api/admin/stats` — Админ статистика
- `GET /api/admin/posts` — Админ посты

## Мидлвер
- Авторизация для `/dashboard/*` и `/admin/*`
- JWT проверка через cookies
- Редирект на `/login` при отсутствии токена

## Безопасность
- Rate limiting (login: 10/мин, register: 5/мин)
- Валидация входных данных (Zod)
- Sanitization HTML input
- Password hashing (bcrypt, 12 раундов)
- JWT tokens (7 дней)
